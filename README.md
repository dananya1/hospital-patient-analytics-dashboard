# Hospital Patient Analytics Dashboard

## Project Overview
This project is a multi-page Hospital Patient Analytics Dashboard developed using Power BI to analyze patient admissions, medical conditions, insurance billing, doctor performance, and patient stay patterns.

## Tools Used
- Power BI
- Power Query
- DAX
- CSV Dataset

## Features
- KPI Cards
- Admission Trend Analysis
- Doctor Performance Analysis
- Insurance Billing Analysis
- Medical Condition Analysis
- Average Stay Analysis
- Interactive Slicers

## Key Insights
- Identified most common medical conditions
- Compared insurance provider billing contributions
- Analyzed patient admission trends
- Evaluated average patient stay duration

## Dashboard Pages
### Page 1: Executive Overview
- KPIs
- Admission trends
- Gender distribution
- Doctor performance

### Page 2: Medical & Financial Insights
- Billing by insurance provider
- Medical condition analysis
- Average patient stay analysis
